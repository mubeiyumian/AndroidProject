# Android环保公益APP

#### 介绍
安卓小应用，是一个关于环保公益的APP，涵盖活动发布参与，积分兑现，勋章查看与网络志愿活动抓取等功能


#### 软件架构
- 开发软件：Android Studio
- API：29（Android 10）


#### 安装教程

1. 复制仓库内green源码
2. 使用Android Studio打开
3. 更换本地SDK路径
4. 自动下载所需要的包，可能有点慢，正常的
5. 源码中涉及了网络数据抓取，自己使用正规软件抓取网络数据存放在asset文件下，命名为output.json
> 如未设置output.json文件，志愿活动打开会是空白
